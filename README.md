# Glumpe AI Agents

Self-evolving AI agent orchestration system built with N8N and MCP.

## Architecture
- **Meta-Agent**: Supervisor that creates and manages other agents
- **N8N Workflows**: Agent definitions and automation
- **MCP Server**: Tool exposure for Claude integration
- **GitHub**: Version control and collaboration

## Agents
- See `/agents` directory for all active agents

## Setup
1. Clone this repo
2. Configure N8N connection
3. Deploy agents via meta-agent
